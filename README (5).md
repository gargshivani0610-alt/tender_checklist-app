# Tender Checklist & Scrutiny – Streamlit

## How to run
1) Install: `pip install -r requirements.txt`
2) Run: `streamlit run app_streamlit.py`

## Files to edit (no coding)
- `config/steps.csv` – steps/labels
- `config/options.csv` – options per step
- `config/suboptions.csv` – one-level child menus
- `config/answers.csv` – standard answers (+ circular mapping)
- `config/circulars.csv` – library of circulars (Title + Link/File)
- `config/rules.csv` – rule expressions and advice
